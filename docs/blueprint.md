# **App Name**: ResumeFlow AI

## Core Features:

- Resume Input Forms: User-friendly form inputs for collecting resume information.
- Template Selection: Template selection feature, users can select resume template from available options.
- AI Summary Suggestion: AI tool for suggesting professional summaries based on user input.
- AI Job Description Rewriter: AI tool to rewrite job descriptions to be more impactful.
- Real-time Resume Preview: Real-time preview of the resume as the user inputs information.
- PDF Export: Download the resume as a PDF file.
- Shareable Resume Link: Generates a unique, shareable URL to resume, for online viewing.

## Style Guidelines:

- Primary color: HSL(210, 67%, 53%) which is a medium blue, converted to hex: #3898FF, evokes a sense of trust and professionalism, fitting for a career-oriented application.
- Background color: HSL(210, 20%, 95%) which is a very light blue gray, converted to hex: #F0F4F7, offers a clean and professional backdrop, ensuring readability and focus on content.
- Accent color: HSL(180, 55%, 50%) which is a teal green, converted to hex: #4DD0E1, to highlight key interactive elements and calls to action, providing a fresh and modern contrast.
- Body and headline font: 'Inter', a sans-serif font that provides a clean and modern look, suitable for both headings and body text. 
- Use modern and minimalist icons to represent different sections and actions, maintaining a clean and intuitive user experience.
- Design a clean and structured layout with clear sections for each part of the resume. Ensure the design is responsive across different screen sizes.
- Subtle animations for interactive elements, such as button hovers and form input transitions, to enhance user engagement.
